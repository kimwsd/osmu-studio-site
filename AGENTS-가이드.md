# OSMÜ STÜDIO — AI 에이전트 사용 가이드

`agency-agents` 232개가 설치돼 있지만, **이 사이트(순수 HTML/CSS/JS 정적 사이트)에 실제로 쓸 만한 건 아래 12개**입니다.
나머지(블록체인·게임개발·중국마케팅·세일즈·금융 등)는 이 프로젝트와 무관하니 신경 쓰지 않아도 됩니다.

## 쓰는 법

채팅에서 **"~한테 시켜줘"** 또는 **"~ 에이전트로 ○○ 해줘"** 라고 말하면 제가 해당 에이전트를 불러서 작업합니다.
이름을 외울 필요 없이 **"무슨 작업"인지만** 말해도 제가 아래 표에서 알맞은 에이전트를 골라 씁니다.

## 가장 자주 쓸 핵심 4개 (이것만 기억해도 충분)

1. **`engineering-frontend-developer`** — "코드 고쳐줘" 류 전부
2. **`design-ui-designer`** — "보기 좋게 다듬어줘" 류 전부
3. **`marketing-content-creator`** — "문구 써줘" 류 전부
4. **`marketing-seo-specialist`** — "검색에 잘 나오게 해줘"

> 💡 사실 이름을 몰라도 됩니다. **"~ 해줘"** 라고만 하면 제가 알아서 맞는 에이전트를 골라 씁니다.
> 이 문서는 "이런 것도 시킬 수 있구나" 참고용입니다.
